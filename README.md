## **neovim_crypt**

A Neovim plugin for encrypting and decrypting files on Linux.  Uses the GPG implementation of AES-256 for encryption.

### Install with [Lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{ "cshmookler/neovim_crypt" },
```

# **neovim_crypt**

A Neovim plugin for encrypting, securely viewing, and decrypting files on Linux.

## Install with [Lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{ "cshmookler/neovim_encrypt", opts = {} },
```
